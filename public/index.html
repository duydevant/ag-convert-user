Cách hoạt động

Flow chuyển đổi user từ Auth0 sang Keycloak

Bước 1: Tạo users test trên Auth0 (hoặc dùng users có sẵn)

Bước 2: Export tất cả users từ Auth0 → Import vào Keycloak (chỉ thông tin, không có password)

Bước 3: Cho users login qua Migration form → Backend xác thực với Auth0 → Nếu đúng → Set password vào Keycloak

✅ Sau khi migrate, user có thể login bằng Keycloak với cùng email/password
