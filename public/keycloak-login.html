Đăng nhập qua Keycloak

Click nút bên dưới để chuyển đến trang đăng nhập Keycloak.
SPI sẽ tự động intercept → xác thực với Auth0 → migrate password → trả về tokens.

ℹ️ Flow hoạt động:
1. Redirect đến Keycloak login UI
2. User nhập username/password
3. Keycloak SPI intercept password → gọi Auth0 validate
4. Auth0 OK → SPI lưu password vào Keycloak + mark migrated
5. Keycloak trả authorization code về FE
6. FE gửi code cho BE (NestJS) → BE exchange lấy tokens
7. Hiển thị cả Keycloak token + Auth0 token (nếu có)
